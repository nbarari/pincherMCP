Added an advisory `Race detector` CI job (#1733 v0.89). CI previously ran the test matrix without `-go test -race`, leaving pincher's concurrent paths — the indexer's per-file goroutines, the SQLite reader pool, the session flusher, the file watcher, the schema-drift watcher — with zero data-race coverage. The new job runs `go test -race ./internal/...` on ubuntu (where a C toolchain is available; the race detector needs cgo, which the job enables for the test build only — the shipped binary stays `CGO_ENABLED=0` pure-Go). It is `continue-on-error: true` (advisory) until a green streak justifies promotion to a required gate, matching the bench-gate promotion discipline.
