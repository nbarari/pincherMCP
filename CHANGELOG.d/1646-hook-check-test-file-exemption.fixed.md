PreToolUse hook now passes through Read calls on test/spec source files (#1646 v0.86). Pre-fix the hook redirected every indexed file >4KB to `context lite=true` — useful for navigation reads, painful for test files because the lite envelope drops the byte content the agent needs to compose an Edit. Test files are written by hand, so every Edit cycle hit the redirect → fallback Read → friction. New `isTestFile` helper matches Go (`*_test.go`), Python (`test_*.py`, `*_test.py`), JS/TS (`*.test.*`, `*.spec.*`), Ruby (`*_spec.rb`, `*_test.rb`), Java (`*Test.java`, `*Tests.java`, `*IT.java`), Swift / Kotlin / C# / PHP equivalents, plus any path under `tests/`, `test/`, `__tests__/`, `spec/`, `e2e/`, `it/` directory segments. Production files at the same size still redirect — exemption is specific. Cross-language matrix test + integration test pin the contract.
