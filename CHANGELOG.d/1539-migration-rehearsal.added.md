**Migration rehearsal CI smoke test (FILE-U, #1539).** `.github/workflows/migration-rehearsal.yml` runs weekly (Wednesday 11:00 UTC) and on `workflow_dispatch`: builds current pincher, downloads the `v0.4.1` release tarball, indexes a deterministic synthetic corpus (Go + YAML + Markdown) with v0.4.1, opens the same data directory with current, and probes `health` / `stats` / `schema` / `search` against the migrated DB. Fails the workflow if any probe returns an error envelope or `search` returns zero results post-migration. Data dir uploaded as artifact (90-day retention) on failure. Methodology documented in [docs/methodology/migration-rehearsal.md](docs/methodology/migration-rehearsal.md). Catches silent migration regressions before #1390 external migration-guide review surfaces them.
