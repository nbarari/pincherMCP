# v1.0 launch artifacts — checklist scaffold

Scaffolding for [#1538](https://github.com/kwad77/pincher/issues/1538) (FILE-T, v0.97). All artifacts ship between v0.97 release-prep and the v1.0 tag push. This file is the predictable single-source-of-truth list so v0.97/v1.0 prep doesn't surface "what did we forget?" questions late.

## Artifacts

Every artifact has:

- A **DRAFT location** — where the placeholder version lives in this repo.
- An **OWNER** — who has the final say on content.
- A **READY-AT** signal — what has to be true for it to be authorable.
- A **PUBLISH STEP** — when/where it ships at v1.0.
